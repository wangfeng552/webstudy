在Linux上安装Git
sudo apt-get install git

安装完成后设置信息
git config --global user.name "Your Name"
git config --global user.email "email@example.com"

创建版本库
git init

从远程库克隆
git clone git@github.com:michaelliao/gitskills.git
git clone http://192.168.1.168/qfx/docs.git
git pull 更新内容
git push 推送内容到库

添加文件到仓库分2步
git add -A 添加目前所有
git add .添加所有
git commit -m "miaoshu" 完成提交
git push origin master	提交到远程分支

git status 	查看状态
git diff 	查看修改的内容

分支管理
git branch		查看分支
git branch <name>	创建分支
git checkout <name>	切换分支
git checkout -b <name>	创建+切换分支
git merge <name> 	合并某分支到当前分支
git branch -d <name>	删除分支

版本回退
HEAD指向当前版本 
git reset --hard commit_id 版本穿梭
git log 		查看历史提交
git log --pretty=oneline最近到最远的提交日志
git reflog		查看命令历史
git checkout -- index.html 丢弃修改回到最近的commit

bug修复
git stash  		隐藏现在的工作区，然后在bug的地方创建分支，修改提交合并
git stash pop		恢复的同时把stash内容也删了
git stash list		查看，就看不到任何stash内容了：

删除文件
git rm text.txt

切换别人的分支
git checkout -b {} origin/{}

查看远程仓库地址
git remote -v
替换远程仓库地址
git remote rm origin
git remote add origin [url]
