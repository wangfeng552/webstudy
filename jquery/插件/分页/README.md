## 使用说明

1. 使用bower安装`bower install jqPaginator`
2. 直接引入`dist`编译好的文件

## 开发说明  
源码基于GruntJS构建，确保安装Node的情况下，在文件目录下运行

    npm install
    
然后运行

    grunt server
    
修改相应代码会自动刷新  
打包编译请运行

    grunt build
    
开发文件位于src目录，编译后的最终文件位于dist目录
