jquery $(document).ready() 与window.onload的区别

0.文档和窗口 $(document) //是当前文档，就是你看到的整个网页 $(window) //当前浏览器的窗口 scroll,resize 1.执行时间 window.onload //必须等到页面内包括图片的所有元素加载完毕后才能执行。 $(document).ready() //是DOM结构绘制完毕后就执行，不必等到加载完毕。 2.编写个数不同 window.onload //不能同时编写多个，如果有多个方法，只执行一个 $(document).ready() //可同时编写多个，都可执行 3.jquery写法 $(window).load() //Window.onload = function (){} $(document).ready() //简写$(function(){})
